# PHP - Adicionar Produto

Projeto de estudo de PHP puro com PDO e MySQL.

## Tecnologias
- PHP 8+
- MySQL
- PDO
- Composer

## Como rodar

1. Clone o repositório
2. Instale as dependências:
   composer install
3. Crie o arquivo .env baseado no .env.example
4. Crie o banco de dados no MySQL:
   CREATE DATABASE pdo;
5. Rode o projeto:
   php index.php
